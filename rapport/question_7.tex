$$G \models ( ( ( p \neq q ) \wedge ( q \sim_{l} p \vee q \sim_{c} p \vee q \sim_{b} p) ) \rightarrow ( x_p \neq x_q )  ) \wedge \bigwedge_{(p,c) \in I} x_p = c$$
